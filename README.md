# Manejo de Excepciones

## Descripción

Esta aplicación de Windows Forms fue desarrollada en C# para practicar el manejo de excepciones y la validación de datos. Permite registrar la información de un empleado mediante el ingreso de nombre, edad y correo electrónico. El sistema verifica que todos los datos sean válidos antes de guardarlos. Si se detecta algún error, se muestra un mensaje explicando el problema y se señala el campo correspondiente mediante un `ErrorProvider`. Además, cuenta con opciones para limpiar los campos y salir de la aplicación de forma segura.

## Características

* Validación de nombre.
* Validación de edad.
* Validación de correo electrónico.
* Uso de `ErrorProvider`.
* Manejo de errores con `try-catch`.
* Mensajes informativos y de advertencia.

## Herramientas Utilizadas

* C#
* Windows Forms
* Regex
* ErrorProvider
